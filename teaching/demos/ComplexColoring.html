Coloring for Complex Functions

An Interactive Applet powered by Sage and MathJax.

(By Kelsey Norman with HTML code from Prof. Gregory V. Bard)

\textrm{This demo displays a coloring of a complex function.} \\ \textrm{Here, we input a function of } z, \textrm{ using } x = real(z) \textrm{ and } y = imag(z). \textrm{ The } x- \textrm{ and } y- \textrm{ dimensions set the viewing window. } \\ \textrm{The modulus is represented by the brightness and the argument is represented by the hue. }
