Practice undetermined coefficients with 1st-order linear ODEs

(By Thomas Scofield)

\textrm{Use this app to develop intuition for the Method of Undetermined Coefficients}\\ \textrm{as applied to first-order linear ODEs in the form } \;y' = a(t)y + f(t).\; \textrm{ You must specify}\\ a(t),\; \textrm{ and pose a form for a particular solution } \;y_p(t).\; \textrm{ The app will apply the linear}\\ \textrm{operator } \;L = \frac{d}{dt} - a(t)\; \textrm{ to the proposed } \;y_p\; \textrm{ which you can compare with } \;f(t).
