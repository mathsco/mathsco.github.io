Is the Function Invertible

(By Thomas Scofield)

\textrm{This app displays the reflection across the line } y=x \textrm{ of those points on the }\newline \textrm{graph of } y = f(x) \textrm{ with } x-\textrm{coordinate in the interval } [a, b]. \textrm{ The graph of } f \textrm{ is green,} \newline \textrm{while the reflected points are purple. Can you make a sketch yourself of the reflected } \newline \textrm{curve before displaying it? Can you anticipate whether } f \textrm{ on the domain } [a, b] \textrm{ is } \newline \textrm{invertible before displaying the reflected points?}
